Github Student Developer Pack

Github offeriert allen Studierenden kostenlos das Student Developer Pack , welches unter anderem Folgendes enthält:

  • Unlimitierte private Repositories auf Github
  • Gratis Continuous Integration mit Travis-CI für private Repositories
  • AWS und Digital Ocean Credits

und vieles mehr!


Für die Registrierung brauchst du:

  • Einen Github Account
  • Dein Github Account muss deine HSR-Email eingetragen haben


Dann kannst du das Paket einfach unter education.github.com/pack beantragen!
