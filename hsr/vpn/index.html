VPN unter Ubuntu einrichten

Hinweis: Richte das VPN nicht im HSR-Secure-WLAN ein - aus dem HSR-Netz können keine VPN-Verbindungen hergestellt werden.

  1. Benötigte Abhängigkeiten installieren
    Bevor du loslegen kannst, musst du noch das Paket network-manager-vpnc-gnome installieren. Jetzt installieren
  2. Öffne die Netzwerkeinstellungen über den Netzwerkindikatoren am oberen linken Bildschirmrand via VPN-Verbindungen + VPN konfigurieren...
  3. Klick auf den Button Hinzufügen
  4. Wähle in der Liste den Eintrag Cisco-kompatibler VPN-Client (vpnc) aus
  5. Weiter geht's mit einem Klick auf Erzeugen...
  6. VPN Einstellungen eintragen - natürlich mit eigenem Benutzernamen & Passwort

    Es sind keine erweiterten Einstellungen und Konfigurationen in anderen Tabs erforderlich.
  7. Speichern mit Klick auf Speichern. Der neue Eintrag erscheint nun in der Liste
  8. Verbindung über den Netzwerkindikatoren am oberen linken Bildschirmrand via VPN-Verbindungen + HSR VPN herstellen


VPN unter macOS einrichten

  1. Öffne die Netzwerkeinstellungen und klicke unten links auf das Plus
  2. Wähle VPN, Cisco IPSec und wähle einen Verbindungsnamen. Klicke anschliessend auf Erstellen.
  3. Gib rechts die Serverdaten ein - natürlich mit eigenem Benutzernamen. Das Passwort kann leer gelassen werden.
    Klicke auf Authentifizierungseinstellungen.
  4. Gib hsrremote als Schlüssel und Gruppenname ein
  5. Optional kannst du das Häckchen setzen bei VPN in der Menüleiste anzeigen. Damit erscheint ein praktisches Icon in der Statusbar, zum schnellen An- und Abmelden.

  6. Nun kannst du dich mit deinem HSR Login anmelden.

VPN einrichten unter anderen Linux Distributionen

Arch Linux

Die Einrichtung via Netzwerk-Manager erfolgt analog zu Ubuntu. Als Abhängigkeit wird das Paket networkmanager-vpnc benötigt, welches sich im extra Repository befindet.

pacman -S networkmanager-vpnc

Fedora

Die Einrichtung via Netzwerk-Manager erfolgt analog zu Ubuntu. Die Abhängigkeit (Paket NetworkManager-vpnc) sollte bereits vorinstalliert sein.
